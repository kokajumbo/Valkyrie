Справочник "Основания увольнения из организации"

Справочник предназначен для хранения списка оснований увольнения работников.

Основание увольнения указывается при оформлении приказа о прекращении (расторжении) трудового договора с работником - увольнении (форма Т-8).

Список оснований увольнения можно заполнить согласно Трудовому кодексу РФ по кнопке Подбор.
