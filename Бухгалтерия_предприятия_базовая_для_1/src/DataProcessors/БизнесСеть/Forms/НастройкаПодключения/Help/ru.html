Синхронизация данных

При изменении прав доступа пользователей информационной базы требуется обновление прав в сервисе 1С:Бизнес-сеть. Обновление производится по команде Синхронизировать.

Отключение от сервиса

Для отключения всех зарегистрированных организаций в информационной базе от сервиса 1С:Бизнес-сеть необходимо выполнить команду Отключиться от сервиса.
При этом, для других участников сервиса, организации будут доступны для обмена документами.
Для полного удаления организаций в сервисе, необходимо в форме регистрации в сервисе выполнить команду Удалить в сервисе (прекратить участие).

Информация для технической поддержки

Отображается информация, которая может быть запрошена службой технической поддержки при расследовании возникающих ошибок:

  • Ключ информационной базы — уникальный ключ информационной базы в сервисе.
  • Ключ пользователя — уникальный ключ пользователя в сервисе.
  • Версия библиотеки электронных документов — текущая версия подсистемы работы с сервисом.
